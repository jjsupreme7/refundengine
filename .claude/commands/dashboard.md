description: Start the Streamlit dashboard
---

Start the RefundEngine dashboard:

1. **Start the dashboard:**
   ```bash
   streamlit run dashboard/Dashboard.py --server.port 5001
   ```

2. **Inform the user:**
   - Dashboard will be available at http://localhost:5001
   - Pages available:
     - Projects: Project management
     - Documents: Document upload/OCR
     - Review Queue: Exception review workflow
     - Claims: Claims drafting
     - Rules: Tax rules browser

3. **Keep running** until user stops it (Ctrl+C)
